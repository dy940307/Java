1. 파일 클래스의 주요 메서드

	String getAbsolutePath()		파일의 절대경로를 문자열로 리턴
	booleanisDirectory()				폴더 여부를 참/거짓으로 리턴
	booleanisFile()							파일 여부를 참/거짓으로 리턴
	String getName()						파일의 이름을 문자열로 리턴
	String getParent()					부모 폴더의 이름을 문자열로 리턴
	String[] list()							경로내의 폴더와 파일 이름을 문자열 배열로 리턴
	File[] listFiles()					경로내의 폴더와 파일 이름을 파일 객체 배열로 리턴
	booleanmkdir()							해당 경로 에폴더 생성( 하위 폴더만 생성 가능 )
	booleanmkdirs()							존재하지 않는 경로상의 모든 폴더 생성
